# Karl He's Personal Web

Personal portfolio site for Karl He, built with React, TypeScript, Vite, Tailwind CSS, and Motion.

## Overview

This website showcases:

- About information
- Projects
- Skills
- Experience
- Contact links

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Motion
- Lucide React

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Production Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

This project is configured for GitHub Pages deployment under:

`https://karlhe2002.github.io/Personal_web/`

The Vite base path is set in `vite.config.ts` for production, so static assets resolve correctly when deployed to the repository subpath.
